# 吵鬧阿比(Noisy Aubrey)

>Line Bot 聊天機器人

### Programming
* Python
    * LineSDK + Flask = 機器人本體
    * BeautifulSoup + Fake UserAgent = 抓取詩籤、卜卦資訊和圖片資源
    * PyExecJS + NodeJS = 抓取執行部分網站JS function
* HeroKu  部署
    * PostgreSQL = 資料庫
    * Scheduler = 定時招呼
